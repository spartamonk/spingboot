# spring-boot-hello

# Pre-requisites:
  - Install Java
  - Install GIT
  - Install Maven
  
# Clone code from github:
    git clone https://github.com/spartamonk/spingboot.git
    cd pring-boot-hello
# Build Maven Artifact:
    mvn clean install
# Deploy springboot application:
    java -jar gs-spring-boot-0.1.0.jar
